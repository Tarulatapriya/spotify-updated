A simple web-based music player built using HTML, CSS, and JavaScript. This app allows users to play unlimited random songs fetched dynamically from the iTunes API, complete with album covers, progress bars, and volume controls.

*Features

Plays random songs using API for a shuffle-like experience.

Shows album cover and song title.

Play, pause, next, and previous controls.

Volume control and progress bar.

Dynamic playlist highlighting.

Fully responsive and visually inspired by Spotify’s interface.

*Technologies Used

HTML5 – Structure of the web page

CSS3 – Styling, gradients, shadows, and responsive design

JavaScript – Audio playback, API fetch, dynamic playlist, and UI interactions

*How to Use

Open index.html in your browser.

Click Play to start listening to random songs.

Use Next/Previous buttons to shuffle songs.

Adjust volume or seek using the progress bar.
